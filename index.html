Block Pronote Picture

Cette extension Firefox vous permet de bloquer ou de remplacer la photo de profil sur Pronote.

Fonctionnalités

  • Bloque la photo de profil sur Pronote
  • Option pour remplacer la photo par une image personnalisée
  • Facile à activer et désactiver

Installation

Pour installer cette extension :

  1. Téléchargez le fichier .xpi depuis cette page
  2. Ouvrez Firefox et allez dans le menu
  3. Sélectionnez "Modules complémentaires et thèmes"
  4. Cliquez sur l'icône d'engrenage et choisissez "Installer un module depuis un fichier"
  5. Sélectionnez le fichier .xpi téléchargé
